# Intelligent Network Security Dashboard

A modern, futuristic single-page React dashboard for a research project on Privacy-Preserving Federated Learning for Industrial IoT.

## Features

- **Dark cybersecurity theme** with glassmorphism and neon accents
- **Real-time pipeline timeline** with animated progress
- **Live metrics panel** with animated counters
- **Federated learning nodes visualization** showing trust scores and status
- **Research pipeline architecture** display
- **Terminal-style activity log** with streaming events
- **Interactive charts** using Recharts
- **Configuration panel** with futuristic sliders

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)
- Recharts (graphs/charts)
- Vite (build tool)

## Installation

```bash
cd dashboard

# Install dependencies
npm install
```

## Running the Application

```bash
# Development server
npm run dev
```

The app will be available at `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` folder.
